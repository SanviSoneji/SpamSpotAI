# 🚀 SpamSpotAI

SpamSpotAI is a multi-feature AI-powered web application built using HTML, CSS, and JavaScript.

It provides an interactive platform for:
- 📝 Text Summarization
- 💬 AI Chatbot Interface
- 🌍 Language Translation
- 🎤 Speech Recognition & Voice Integration
- 🔐 User Authentication System
- 📊 Dashboard & Settings Panel

---

## 🌐 Live Demo
👉 https://sujalarora17.github.io/SpamSpotAI/

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Web APIs (Speech API, Clipboard API)
- LocalStorage (Client-side Authentication)

---

## ✨ Key Features

- Modular page architecture
- Responsive and clean UI
- Chat history handling
- Speech-to-text integration
- Word counter & clipboard functionality
- Dashboard-based navigation
- Session handling using LocalStorage
